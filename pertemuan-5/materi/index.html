1. List itu apa? 📝

List dipakai untuk menulis data berurutan atau poin-poin.

Jenis list di HTML:

  • <ul> = bullet list
  • <ol> = list bernomor
  • <li> = isi tiap item

2. Unordered List (UL) •


<ul>
  <li>Pensil</li>
  <li>Buku</li>
  <li>Penghapus</li>
</ul>

Hasilnya: titik bullet di depan item.

3. Ordered List (OL) 1,2,3


<ol>
  <li>Bangun pagi</li>
  <li>Sarapan</li>
  <li>Berangkat sekolah</li>
</ol>

Hasilnya: item otomatis bernomor.

4. Tabel itu apa? 📊

Tabel dipakai untuk data yang punya baris dan kolom.


<table>
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Rani</td>
    <td>11</td>
  </tr>
</table>

Kapan Pakai List vs Tabel?

Pakai Listuntuk poin singkat, ide, langkah kegiatan.

Pakai Tabeluntuk data yang dibandingkan per kolom.

Contoh Listhobi, alat sekolah, jadwal harian.

Contoh Tabelnama, umur, nilai, hobi.

Bedah Tag Tabel 🔍

  • <table> = pembungkus tabel
  • <tr> = table row (baris)
  • <th> = judul kolom
  • <td> = isi data

Contoh Tabel Nilai 🏆

Nama Nilai HTML
Alya 95
Bima 90

Tips List & Tabel 💡

  • Gunakan list untuk poin singkat
  • Gunakan tabel untuk data perbandingan
  • Jangan isi tabel terlalu banyak kolom
  • Beri judul kolom yang jelas

Langkah Praktik Bertahap

  1. Buat judul halaman: Profil Kelas Saya.
  2. Tambah UL berisi 3 hobi favorit.
  3. Tambah OL berisi 3 kegiatan pagi.
  4. Buat tabel 3 kolom dan isi minimal 3 baris data.
  5. Rapikan tampilan dengan border, padding, dan warna.

🚀 Misi Hari Ini

Buat halaman profil kelas berisi:

  1. List hobi favorit (pakai <ul>)
  2. List jadwal harian (pakai <ol>)
  3. Tabel data teman (Nama, Umur, Hobi)

Troubleshooting Cepat

  • Nomor OL tidak muncul: pastikan pakai <ol>, bukan <ul>.
  • Tabel berantakan: cek setiap baris punya jumlah <td> yang sama.
  • Border tidak terlihat: tambahkan style border di table/th/td.
  • Data tidak sejajar: periksa tag <tr> terbuka dan tertutup dengan benar.
