Cheat Sheet: List & Tabel

Alur Cepat Belajar Hari Ini

  1. Kenali perbedaan UL, OL, dan Tabel
  2. Latihan membuat 2 jenis list
  3. Latihan membuat tabel 3 kolom
  4. Cek kerapian HTML
  5. Presentasi singkat hasil halaman

Tag List Dasar

Tag Fungsi Contoh
<ul> List bullet <ul>...</ul>
<ol> List bernomor <ol>...</ol>
<li> Item list <li>Item</li>

Contoh UL dan OL

<ul> <li>Membaca</li> <li>Menggambar</li> </ul> <ol> <li>Bangun tidur</li> <li>Sarapan</li> </ol>

Template Paling Aman

<h1>Profil Kelas Saya</h1> <h2>Hobi Favorit</h2> <ul> <li>Membaca</li> <li>Bermain bola</li> <li>Menggambar</li> </ul> <h2>Jadwal Pagi</h2> <ol> <li>Bangun tidur</li> <li>Sarapan</li> <li>Berangkat sekolah</li> </ol>

Tag Tabel Dasar

Tag Fungsi Contoh
<table> Wadah tabel <table>...</table>
<tr> Baris <tr>...</tr>
<th> Header kolom <th>Nama</th>
<td> Data kolom <td>Alya</td>

Contoh Tabel

<table> <tr> <th>Nama</th> <th>Umur</th> </tr> <tr> <td>Rani</td> <td>11</td> </tr> </table>

Snippet Tabel Lebih Rapi

<table border="1" cellpadding="6" style="border-collapse:collapse;"> <tr> <th>Nama</th> <th>Umur</th> <th>Hobi</th> </tr> <tr> <td>Alya</td> <td>11</td> <td>Membaca</td> </tr> </table>

Tips Cepat

  • Pakai UL untuk daftar tanpa urutan.
  • Pakai OL untuk langkah berurutan.
  • Pakai TH untuk judul kolom tabel.
  • Data yang sejenis lebih mudah dibaca jika pakai tabel.

Troubleshooting Cepat

Masalah Penyebab Solusi
Nomor list tidak muncul Pakai UL, bukan OL Ubah tag menjadi <ol>
Tabel miring/acak Jumlah kolom tidak sama Samakan jumlah <td> di tiap baris
Border tabel hilang Style border belum ada Tambahkan border pada table, th, dan td

Checklist Final

  • UL berisi minimal 3 item
  • OL berisi minimal 3 langkah
  • Tabel punya header yang jelas
  • Tabel berisi minimal 3 baris data
  • Semua tag pembuka dan penutup sudah benar

Tips presentasi: jelaskan mengapa kamu memilih data tersebut dan bagian mana yang paling kamu banggakan.
