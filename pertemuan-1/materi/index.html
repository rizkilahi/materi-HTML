üìê Struktur Wajib HTML

Setiap membuat web, harus ada kode ini:


<!DOCTYPE html>
<html>
  <head>
    <title>Judul di Tab Atas</title>
  </head>
  <body>
    <!-- Isi website di sini -->
  </body>
</html>
  • <html> : Rumah besar yang bungkus semua
  • <head> : Kepala/otak (info website)
  • <body> : Badan (isi yang terlihat)

üíæ Tips: Simpan file dengan nama index.html
