üìù Cheat Sheet: Link & Navigasi

üîó 1. Tag <a> (Anchor/Link)

Sintaks Dasar:

<a href="alamat-tujuan">Teks yang diklik</a>
Komponen Penjelasan Contoh
<a> Tag pembuka link <a href="...">
href Atribut untuk alamat tujuan href="google.com"
Teks Link Teks yang akan diklik Klik di sini
</a> Tag penutup link </a>

üåê 2. Jenis-Jenis Link

A. Link External (ke website lain)

Contoh:

<a href="https://youtube.com">Pergi ke YouTube</a> <a href="https://google.com">Buka Google</a> <a href="https://wikipedia.org">Wikipedia</a>
üí° Tips: Link external harus menggunakan https:// atau http://

B. Link Internal (ke halaman sendiri)

Contoh:

<a href="halaman2.html">Ke Halaman 2</a> <a href="tentang.html">Tentang Saya</a> <a href="kontak.html">Kontak</a>
üí° Tips: Link internal cukup tulis nama file tanpa https://

üéØ 3. Atribut target

Atribut Fungsi Contoh Kode
target="_blank" Buka di tab baru <a href="google.com" target="_blank">Google</a>
target="_self" Buka di tab yang sama (default) <a href="page.html" target="_self">Page</a>

Contoh Penggunaan:

<!-- Buka di tab baru --> <a href="https://youtube.com" target="_blank">YouTube (Tab Baru)</a> <!-- Buka di tab yang sama --> <a href="halaman2.html">Halaman 2 (Tab Sama)</a>

üß≠ 4. Membuat Menu Navigasi

Cara 1: Menu Sederhana

<nav> <a href="index.html">Home</a> | <a href="about.html">About</a> | <a href="contact.html">Contact</a> </nav>

Cara 2: Menu dengan Styling

<nav style="background: lightblue; padding: 15px; text-align: center;"> <a href="index.html" style="margin: 0 10px; color: blue;">Home</a> <a href="about.html" style="margin: 0 10px; color: blue;">About</a> <a href="contact.html" style="margin: 0 10px; color: blue;">Contact</a> </nav>

Cara 3: Menu dengan List

<nav> <ul> <li><a href="index.html">Home</a></li> <li><a href="about.html">About</a></li> <li><a href="contact.html">Contact</a></li> </ul> </nav>

üé® 5. Styling Link dengan CSS

Properti Fungsi Contoh Nilai
color Warna teks link blue, red, #11998e
text-decoration Garis bawah link none, underline
background Warna latar belakang lightblue, yellow
padding Jarak dalam 10px, 15px 20px
border-radius Sudut membulat 5px, 10px

Contoh Styling Link:

<!-- Link dengan button style --> <a href="page.html" style="background: blue; color: white; padding: 10px 20px; text-decoration: none; border-radius: 5px;"> Klik Saya </a> <!-- Link dengan hover effect (pakai CSS) --> <style> a:hover { color: red; background: yellow; } </style>

üí° 6. Tips & Trik

‚úÖ DO (Lakukan):
  • Gunakan teks link yang jelas dan informatif
  • Pakai target="_blank" untuk link external
  • Buat menu navigasi yang konsisten di semua halaman
  • Test semua link sebelum dipublish
‚ùå DON'T (Jangan):
  • Jangan pakai teks "Klik di sini" saja
  • Jangan lupa tutup tag </a>
  • Jangan lupa tanda kutip di atribut href
  • Jangan pakai link yang rusak (broken link)
