Gambar, Warna & Gaya Tulisan

üñºÔ∏è 1. Memasukkan Gambar

Tag: <img>

Contoh Kode: 
<img src="kucing.jpg" width="300px">
Atribut Kegunaan Contoh
src Nama/path file gambar src="kucing.jpg"
width Lebar gambar width="300px"
height Tinggi gambar height="200px"
alt Teks alternatif jika gambar tidak muncul alt="Foto Kucing"
‚ö†Ô∏è Penting:
  • File gambar harus ada di folder yang sama dengan file HTML
  • Perhatikan nama file (huruf besar/kecil harus sama)
  • Tag <img> tidak perlu penutup!

üåà 2. Memberi Warna Tulisan

Atribut: style="color:namaWarna"

Contoh Kode: 
<h1 style="color:red">Halo Merah!</h1>
<p style="color:blue">Paragraf biru.</p>
Hasilnya:

Halo Merah!

Paragraf biru.

Warna Nama Inggris Contoh
Merah red color:red
Biru blue color:blue
Hijau green color:green
Oranye orange color:orange
Ungu purple color:purple
Pink pink color:pink
Kuning yellow color:yellow
Coklat brown color:brown

üñåÔ∏è 3. Warna Latar Belakang

Atribut: style="background-color:namaWarna"

Contoh untuk tulisan: 
<h1 style="background-color:yellow">Stabilo Kuning</h1>
Hasilnya:

Stabilo Kuning

Contoh untuk seluruh halaman: 
<body style="background-color:lightblue">
  <h1>Website Saya</h1>
  <p>Latar belakang halaman biru muda.</p>
</body>
üí° Warna Background Populer:
  • lightblue - Biru muda
  • lightgreen - Hijau muda
  • lightyellow - Kuning muda
  • lightpink - Pink muda
  • lavender - Ungu muda

‚ú® 4. Gaya Tulisan (Formatting)

Gaya Tag Contoh Kode Hasil
Tebal (Bold) <b></b> <b>tebal</b> tebal
Miring (Italic) <i></i> <i>miring</i> miring
Garis Bawah <u></u> <u>garis</u> garis
Contoh Kombinasi: 
<p>Saya <b>sangat</b> suka makan <i>Nasi Goreng</i>!</p>
<p><u>Ini digarisbawahi</u> dan <b><i>ini tebal miring</i></b>.</p>
Hasilnya:

Saya sangat suka makan Nasi Goreng!

Ini digarisbawahi dan ini tebal miring.

üí° Tips & Trik

‚úÖ Yang Perlu Diingat:
  1. Simpan gambar di folder yang sama dengan file HTML
  2. Nama file gambar harus persis sama (huruf besar/kecil)
  3. Gunakan width agar gambar tidak terlalu besar
  4. Tutup semua tag yang dibuka (kecuali <img> dan <br>)
  5. Kombinasikan warna dan gaya untuk hasil menarik!
‚ùå Kesalahan Umum:
  • Lupa menulis px di width="300px"
  • Salah ketik nama file gambar
  • Gambar tidak di folder yang sama
  • Lupa tanda kutip di atribut src=""
