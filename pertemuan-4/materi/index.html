🎯 Mini Project dan Presentasi

Kursus HTML Pertemuan 4 (2 Jam)

Waktunya menyelesaikan website sederhana kamu sendiri!

Tujuan Hari Ini

  • Anak menyelesaikan 1 website sederhana.
  • Anak percaya diri mempresentasikan hasil karya.
  • Anak belajar cek hasil sendiri sebelum dipresentasikan.

Rundown 2 Jam ⏰

Durasi Kegiatan Target
20 menit Review cepat HTML Ingat tag penting
60 menit Bangun mini project Website jadi dan rapi
30 menit Presentasi karya Percaya diri berbicara
10 menit Apresiasi dan foto karya Penutup menyenangkan

Review Semua Tag HTML 🔁

  • <h1>, <p> untuk judul dan teks
  • <img> untuk gambar
  • <a> untuk link dan navigasi
  • <ul>, <ol>, <table> untuk data

Aturan Mini Project

WajibJudul + 2 paragraf + 1 gambar + 1 link

WajibStruktur HTML rapi dan tag ditutup

BonusTambah list atau tabel sederhana

BonusWarna tema sesuai topik website

Rapikan Struktur HTML 🧼

  • Pastikan ada <!doctype html>
  • Gunakan <head> dan <body> dengan benar
  • Urutkan konten: judul, isi, penutup
  • Gunakan indentasi yang rapi

Sentuhan Akhir ✨

  • Tambah warna pada judul dan background
  • Tambah minimal 1 gambar
  • Buat tampilan lebih personal sesuai tema

Mini Project (Bebas Pilih)

  1. Website Tentang Aku
  2. Website Hewan Kesukaan
  3. Website Game Favorit
  4. Website Keluargaku

Contoh Ide Isi per Tema

Tentang Aku
Nama, hobi, makanan favorit

Hewan Kesukaan
Jenis hewan, habitat, fakta lucu

Game Favorit
Nama game, alasan suka, tips bermain

Keluargaku
Anggota keluarga, aktivitas bersama

Checklist Project ✅

  • Ada judul website
  • Ada isi utama (minimal 2 paragraf)
  • Ada gambar
  • Ada warna
  • Ada minimal 1 link

Kesalahan Umum dan Solusinya

  • Gambar tidak muncul: cek nama file gambar sama persis.
  • Link tidak jalan: pastikan ada https://.
  • Layout berantakan: cek tag yang belum ditutup.
  • Teks terlalu mepet: tambah jarak pakai margin atau br.

Rubrik Penilaian Sederhana

Aspek Skor 1 Skor 2 Skor 3
Kelengkapan isi Kurang lengkap Cukup Lengkap
Kerapian HTML Masih acak Cukup rapi Rapi sekali
Presentasi Masih malu Cukup jelas Percaya diri

Sesi Presentasi 🎤

Anak menjelaskan:

  • Judul web
  • Isi web
  • Bagian favorit

Format Presentasi 30-60 Detik

  1. Halo, nama saya ... dan ini website saya.
  2. Website ini bercerita tentang ...
  3. Bagian favorit saya adalah ...
  4. Terima kasih sudah melihat karya saya.

Penutup Seru 🎉

  • Sertifikat mini
  • Foto hasil karya
  • Apresiasi untuk semua peserta
  • Tantangan rumah: tambahkan 1 halaman baru
