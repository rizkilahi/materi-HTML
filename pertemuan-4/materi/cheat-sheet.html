Cheat Sheet: Mini Project & Presentasi

Alur Cepat Pertemuan 4

  1. Pilih tema project
  2. Tulis rencana isi singkat
  3. Buat halaman HTML lengkap
  4. Cek error dan rapikan tampilan
  5. Presentasi 30-60 detik

Tag Inti yang Wajib Dipakai

  • <h1> untuk judul website
  • <p> untuk isi cerita
  • <img> untuk gambar
  • <a> untuk link
  • style untuk warna sederhana

Template Kerangka Paling Aman

<!doctype html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Judul Project</title> </head> <body> <h1>Judul Website</h1> <p>Paragraf pembuka</p> <img src="gambar.jpg" width="220" alt="deskripsi"> <p>Paragraf isi kedua</p> <a href="https://example.com" target="_blank">Referensi</a> </body> </html>

Template Cepat Halaman Project

<!doctype html> <html> <head> <title>Project Saya</title> </head> <body style="background:#fff7ed;"> <h1 style="color:#c2410c;">Judul Website Saya</h1> <p>Ini adalah cerita singkat tentang tema website saya.</p> <img src="gambar.jpg" width="220" alt="gambar tema"> <p>Bagian ini adalah isi favorit saya.</p> <a href="https://example.com" target="_blank">Link referensi</a> </body> </html>

Snippet Bonus: List dan Tabel

<h2>Fakta Singkat</h2> <ul> <li>Fakta 1</li> <li>Fakta 2</li> </ul> <h2>Data Kecil</h2> <table border="1" cellpadding="6"> <tr><th>Nama</th><th>Info</th></tr> <tr><td>Item 1</td><td>Deskripsi</td></tr> </table>

Palet Warna Rekomendasi

  • Latar hangat: #fff7ed
  • Judul utama: #c2410c
  • Subjudul: #b91c1c
  • Kotak info: #ffedd5

Checklist Final Sebelum Presentasi

  1. Judul sudah jelas
  2. Isi minimal 2 paragraf
  3. Gambar tampil dengan benar
  4. Warna sudah ditambahkan
  5. Link bisa diklik
  6. Tag pembuka dan penutup sudah cocok
  7. Tidak ada teks yang terlalu kecil

Troubleshooting Cepat

Masalah Penyebab Umum Solusi Cepat
Gambar tidak tampil Nama file tidak sama Samakan nama di src dengan file asli
Link tidak terbuka URL belum lengkap Tambahkan https:// di depan alamat
Tampilan berantakan Ada tag belum ditutup Cek ulang pasangan tag pembuka-penutup

Panduan Presentasi Anak

  1. "Judul web saya adalah ..."
  2. "Isi web saya tentang ..."
  3. "Bagian favorit saya adalah ..."

Tips berbicara: lihat audiens, bicara perlahan, dan sebutkan 1 hal yang paling dibanggakan dari project.

Penutup Kelas

  • Sertifikat mini untuk semua peserta
  • Foto hasil karya bersama
  • Tantangan rumah: tambah 1 halaman baru dengan link navigasi
