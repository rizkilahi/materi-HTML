Pilih 1 tema mini project:

  • Website Tentang Aku
  • Website Hewan Kesukaan
  • Website Game Favorit
  • Website Keluargaku

Isi paragraf 2: ____________________

Bagian favorit: ____________________

Langkah 2: Struktur Website Lengkap

<!doctype html> <html lang="id"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>[Judul Website]</title> </head> <body style="background:#fff7ed; font-family:Arial,sans-serif;"> <h1 style="color:#c2410c;">[Judul Website]</h1> <p>[Paragraf pembuka tentang tema kamu]</p> <img src="[nama-gambar].jpg" width="220" alt="[deskripsi gambar]"> <p>[Paragraf kedua tentang hal favoritmu]</p> <h2>Fakta Singkat</h2> <ul> <li>Fakta 1</li> <li>Fakta 2</li> <li>Fakta 3</li> </ul> <a href="https://contoh.com" target="_blank">Link Referensi</a> </body> </html>

Langkah 3: Opsi Bonus (Pilih Salah Satu)

  • Tambah tabel kecil berisi 3 data penting
  • Tambah halaman kedua dan hubungkan dengan link
  • Tambah 2 gambar dan beri keterangan singkat

Checklist Sentuhan Akhir

  • Tambahkan warna judul dan background
  • Tambahkan minimal 1 gambar
  • Rapikan jarak antar elemen
  • Cek semua tag sudah ditutup
  • Pastikan link bisa dibuka
  • Pastikan ukuran teks nyaman dibaca

Cek Error Cepat Sebelum Presentasi

  • Jika gambar kosong: cek nama file dan format .jpg/.png
  • Jika link error: cek penulisan https://
  • Jika tampilan berantakan: cek tag yang belum ditutup
  • Jika tulisan menumpuk: tambah baris baru atau margin
